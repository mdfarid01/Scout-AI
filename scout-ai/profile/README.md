# Your Profile

Put your candidate profile here as one or more `.md` files. Everything in this
folder is concatenated and given to the matching and outreach agents.

Suggested files:

- `resume.md` — your resume in markdown (experience, projects, skills)
- `preferences.md` — what you're looking for (role, stack, location, stage)

Example `preferences.md`:

```markdown
## What I'm looking for
- Role: Full-stack / backend engineer
- Stack: MERN, Python, AWS
- Stage: Seed to Series B
- Location: Remote or Bangalore
- Notable projects: RapidTrack (fintech), NFTinema (web3), Chrome extension (AI)
```
